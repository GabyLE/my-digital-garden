## Histogram^[https://seaborn.pydata.org/generated/seaborn.histplot.html]
- Represents the distribution of one or more variables.
- Returns matplotlib axes containing the plot.
```python
seaborn.histplot(data=, x=, y=, bins=, kde=)
```
