## `seaborn.set_style(style=, rc=)`^[https://seaborn.pydata.org/generated/seaborn.set_style.html]

- Control general style of the plots, like:
	- color of the background
	- enabled grid
- Style options:
	- `whitegrid`
	- `white`
	- `dark`
	- `darkgrid`
	- `ticks`
	- Dictionary of parameters
## Basics
```python
sns.set_title()
sns.set_xlabel()
sns.set_ylabel()
```
